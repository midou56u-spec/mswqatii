# مسوقاتي AI - تشغيل التطبيق محلياً

عشان تشغل التطبيق ده على جهازك (سيرفر محلي)، اتبع الخطوات دي:

## المتطلبات
1. تثبيت **Node.js** (إصدار 18 أو أحدث) من [nodejs.org](https://nodejs.org/).

## خطوات التشغيل
1. **تحميل الكود**: حمل ملفات المشروع على جهازك.
2. **فتح التيرمينال**: افتح الـ Terminal أو Command Prompt في فولدر المشروع.
3. **تثبيت المكتبات**: اكتب الأمر ده واستنى لما يخلص:
   ```bash
   npm install
   ```
4. **إعداد مفاتيح الربط**:
   - هتلاقي ملف اسمه `.env.example`.
   - غير اسمه لـ `.env`.
   - حط مفتاح الـ Gemini API بتاعك قدام `GEMINI_API_KEY`.
5. **تشغيل السيرفر**: اكتب الأمر ده:
   ```bash
   npm run dev
   ```
6. **فتح التطبيق**: افتح المتصفح وادخل على الرابط اللي هيظهرلك (غالباً هيكون `http://localhost:3000`).

## ملاحظات هامة
- التطبيق بيستخدم **Firebase**، فتأكد إن ملف `firebase-applet-config.json` موجود وفيه بيانات الربط الصحيحة.
- لو واجهت أي مشكلة في الصلاحيات، تأكد إنك رفعت قواعد البيانات (Firestore Rules) على حساب الفايربيز بتاعك.
